# Obsidian 万年历

在新标签页中显示中国万年历**年视图**（4×3 自适应网格），含农历、二十四节气、公历与农历节日。

## 功能

- 左侧功能区图标或命令面板 →「打开万年历」
- **年视图**：自适应网格（宽屏 4 列），左右边距 20px，月历间距 10px，月历拉满可用宽度
- 悬停日期显示农历、干支与节日详情
- 上一年 / 下一年 / 今年 快速切换
- **黄历详情**：点击任意日期，居中弹出当日黄道吉日（宜/忌）、二十四节气与节日、公历农历干支；左右箭头切换日期
- **自定义事件**：按自定义标签页分类（默认内置节假日、生日等可自建）；顶部工具栏进入管理

## 开发

```bash
npm install
npm run dev
npm run build
```
